Diseño responsivo y accesibilidad web

El diseño responsivo garantiza que una página web se adapte a distintos tamaños de pantalla y dispositivos. La accesibilidad busca que las páginas sean utilizables por la mayor cantidad de personas posible, incluyendo personas con discapacidades.

Principios

Usar unidades relativas, rejillas fluidas y consultas de medios (media queries) para adaptar el diseño. Priorizar la legibilidad, contraste y una estructura semántica clara.

Accesibilidad

Incluir texto alternativo en imágenes, etiquetas en formularios, y un orden lógico de lectura. Garantizar contraste suficiente y navegación con teclado.

Buenas prácticas

Usar HTML semántico, evitar dependencias de JS para funciones básicas (como la navegación principal) y proporcionar estructuras de contenido claras.

Recursos y objetivos de aprendizaje

  • Entender cómo funcionan las media queries y unidades relativas.
  • Aprender técnicas para mejorar la accesibilidad.
  • Aplicar estos conceptos en proyectos web prácticos.
